# Gestion des Prospects

## Vue d'ensemble

Le module de gestion des prospects est une composante centrale du CRM MaBoussole. Il permet de gérer efficacement le cycle de vie complet des prospects, de leur entrée initiale jusqu'à leur conversion en clients.

## Fonctionnalités principales

- Création et gestion des prospects avec informations détaillées
- Système de statuts avec codes couleur (Nouveau, En analyse, Approuvé, Refusé, Converti)
- Actions individuelles et en masse
- Conversion de prospects en clients
- Système d'assignation et de partenariat

## Structure technique

- **Model** : `App\Models\Prospect`
- **Resource** : `App\Filament\Resources\ProspectResource`
- **Interface** : Filament Admin Panel
